# 家居好物推荐网站

完全自动化的淘宝联盟赚钱网站，通过推荐高性价比家居好物获得佣金收入。

## 网站特点

### 🎯 核心功能
- 家居好物推荐和评测
- 淘宝联盟商品推广
- SEO优化获取自然流量
- 完全自动化内容更新

### 🚀 技术栈
- **静态网站**：HTML5 + CSS3 + JavaScript
- **托管**：GitHub Pages（免费）
- **自动化**：GitHub Actions
- **分析**：Google Analytics（可选）

### 💰 收入模式
1. 通过优质内容吸引用户
2. 推荐高佣金家居商品
3. 用户点击链接购买
4. 获得淘宝联盟佣金

## 部署指南

### 快速部署
1. 将本文件夹所有文件上传到GitHub仓库
2. 启用GitHub Pages
3. 网站立即上线

### GitHub Pages设置
1. 进入仓库 Settings
2. 选择 Pages
3. Source: 选择 `main` 分支
4. 保存后等待部署完成
